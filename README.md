# personal-portfolio-2.0

## Installation
### Linux
```bash
pip install virtualenv
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```
### Windows
```bash
pip install virtualenv
python -m virtualenv venv
venv\scripts\activate
pip install -r requirements.txt
```

## Run
### Linux
```bash
export FLASK_ENV=development
flask run
```
### Windows
```bash
set FLASK_ENV=development
flask run
```
